This is simple responsive and fluid application layout, built upon jQuery,bootstrap and animate.css, tested also in hybrid android mobile app.

Layout functionality

javascript variables (you may change them: see szAppConfig.js)
  • szLayOut_wideScreen = 1400; (used for wide screen)
  • szLayOut_middleScreen = 1100; (between 1100 and 1400: used for middle width screen, under 1100 : used for small screen)
  • szLayOut_SideNavWidth = 350; (default widths for side navs)
  • szLayOut_scrollUp = 100; (scroll up button will be shown when user scrols ower this value and hidden again under it)
  • szLayOut_diveToHide = 500; (top menu will be hidden when user scrolles ower this value and shown again under it)
  • szLayOut_headerheight: 57; (header height)
  • Font sizes
    • szLayOut_wideScreenFs: 1rem
    • szLayOut_middleScreenFs: 1.1rem
    • szLayOut_smallScreenFs: 1.2rem
Scroll line
Scroll line is show at the bottom of the page and incicates current vertical position.
Touch support
We are using hammer.js to support touch gestures. Right menu will be shown on swipe from left (only on scrren width under 1200px) and hiden on swipe from right.
